
### Summary

* from [this tutorial](https://github.com/kiteco/kite-python-blog-post-code/tree/master/flask-tutorial/restful-apis-with-flask)
* `pipenv install`
* `pipenv shell`
* to start web server:
    - `./run_server.sh`
    - browse to http://0.0.0.0:4996/books
* to run client:
    - `./run_client.sh`
