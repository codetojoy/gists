
### Summary

* `pipenv install`
* `pipenv shell`
* to start web server:
    - `./run_server.sh`
* to run client in another terminal:
    - `./run_client.sh` 

### Notes

* using `Pipfile` is different from the Flask documentation, but it seems to be OK 
* alternate steps:

```
python3 -m venv venv
. venv/bin/activate
pip install Flask
pip install sqlalchemy 
```
